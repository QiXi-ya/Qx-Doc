鼠标消息

QMsg::MouseMsg 类 - 鼠标事件处理

Mouse::Button 枚举

在 Qx 图形库中，鼠标按钮通过 Mouse::Button 枚举表示。这个枚举位于 SFML 图形库中，提供了标准的鼠标按钮标识。

枚举值 描述 对应物理按钮
Mouse::Button::Left 鼠标左键 主点击按钮，用于选择和拖拽
Mouse::Button::Right 鼠标右键 辅助按钮，用于上下文菜单
Mouse::Button::Middle 鼠标中键（滚轮按钮） 按下滚轮按钮，用于特殊操作
Mouse::Button::XButton1 侧键1（前进键） 部分鼠标的附加按钮
Mouse::Button::XButton2 侧键2（后退键） 部分鼠标的附加按钮

位置获取函数

GetMousePosWindow
静态
获取鼠标在指定窗口上的位置。返回相对于窗口客户区左上角的坐标。坐标以像素为单位，类型为 Vector2i（包含 x 和 y 成员）。
static Vector2i GetMousePosWindow ( RenderWindow& window )

参数

window RenderWindow& 目标窗口引用，获取相对于此窗口的鼠标位置
GetMousePosScreen
静态
获取鼠标在屏幕上的绝对位置。返回相对于屏幕左上角的坐标，不考虑任何窗口。坐标以像素为单位，类型为 Vector2i。
static Vector2i GetMousePosScreen ()

状态检测函数

IsMouseDown
静态
判断当前鼠标按钮是否被按下。检测指定鼠标按钮是否在当前帧被按下（按下事件）。这个函数只在按钮刚刚按下时返回 true，适合检测点击事件。
static bool IsMouseDown ( RenderWindow& Dest, Mouse::Button button, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用，检测此窗口的鼠标事件
button Mouse::Button 要检测的鼠标按钮（左键、右键、中键等）
id int 消息ID，-1表示使用默认消息ID
IsMouseIn
静态
判断鼠标是否在指定区域内。检测当前鼠标是否位于窗口的指定矩形区域内。常用于实现按钮悬停效果、区域选择等交互功能。
static bool IsMouseIn ( int x, int y, int w, int h, RenderWindow& Dest, int id = -1 )

参数

x, y int 矩形区域的左上角坐标（窗口坐标）
w, h int 矩形区域的宽度和高度（像素）
Dest RenderWindow& 目标窗口引用
id int 消息ID，-1表示使用默认消息ID
IsMousePress
静态
判断鼠标按钮是否处于按下状态。检测指定鼠标按钮是否被持续按下（按住状态）。只要按钮处于按下状态，此函数就返回 true，适合实现拖拽、绘制等持续操作。
static bool IsMousePress ( RenderWindow& Dest, Mouse::Button button, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用，检测此窗口的鼠标事件
button Mouse::Button 要检测的鼠标按钮（左键、右键、中键等）
id int 消息ID，-1表示使用默认消息ID

滚轮和移动函数

getMouseWheel
静态
获取鼠标滚轮值。返回鼠标滚轮的当前值，通常是 120 的倍数。正值表示向上滚动，负值表示向下滚动。在每次获取后，滚轮值会被重置。
static int getMouseWheel ()
IsWheel
静态
判断鼠标滚轮是否被滚动。检测当前帧是否有滚轮滚动事件发生。这个函数只在滚轮刚刚滚动时返回 true，适合检测单次滚轮事件。
static bool IsWheel ()
IsMouseMove
静态
判断鼠标是否移动。检测当前帧鼠标是否发生了移动。可以指定消息ID来检测特定窗口的鼠标移动事件，常用于优化性能（只在鼠标移动时更新相关逻辑）。
static bool IsMouseMove ( int id = -1 )

参数

id int 消息ID，-1表示使用默认消息ID

触摸支持函数

GetTouchNum
静态
获取当前触摸点数。返回当前屏幕上正在进行的触摸点数量。支持触摸屏设备，可以用于实现多点触控交互。返回值为当前活动的触摸点数量。
static int GetTouchNum ()
