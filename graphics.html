图形绘制 API

QGraph 类 - 基础图形绘制、图片操作和文字显示

QGraph 类

静态类

QGraph 是一个功能丰富的图形绘制类，包含多个子类用于处理不同类型的图形操作。 该类提供了从基础几何图形绘制到复杂图片处理和文字显示的全套功能。所有方法都是静态的，无需创建类实例即可使用。 包含 Line、Circle、Rectangle 等子类用于绘制基本图形，QIMAGE 和 QIMAGEEX 用于图片操作，QTEXT 用于文字显示。

颜色与基础图形

提供基础的图形绘制功能，包括线条、圆形、椭圆、矩形等基本几何图形的绘制。支持设置颜色、线宽、线型和填充效果。
