消息处理 API

QMsg 类 - 鼠标、键盘和窗口消息处理

QMsg 类

静态类

QMsg 是一个功能完整的消息处理类，专门用于处理窗口消息、鼠标输入和键盘输入。该类采用多子类结构，将不同功能模块化，便于使用和维护。支持多窗口消息管理、消息ID系统和高级输入处理。包含基础消息更新功能、鼠标消息处理、键盘消息处理和复合功能处理。

底层函数

提供消息处理的基础功能，包括消息更新、窗口关闭判断、消息ID管理和缓冲区清理。这些函数是消息系统运行的基础。

鼠标消息

提供鼠标事件处理功能，包括鼠标位置获取、按钮状态检测、滚轮控制和区域判断。支持多窗口和多鼠标按钮。

键盘消息
