图片操作 API

QImage 和 QImageEx 类 - 图片加载、处理和显示

IMAGE 结构体

struct IMAGE {
  Texture texture;
  Color color;
};

IMAGE 结构体是图片操作的核心数据结构，包含两个成员：

  • texture: 存储图片的纹理数据，用于实际的渲染和显示
  • color: 图片的颜色滤镜，可以叠加颜色效果到图片上

QImage 类 - 基础图片操作

newimage
静态
从文件中创建图片对象。支持多种图片格式（PNG、JPG、BMP等），可以指定从图片中截取特定区域。如果加载失败返回false。
static bool newimage ( IMAGE& img, const string& filename, int x = 0, int y = 0, int w = 0, int h = 0 )

参数

img IMAGE& 输出的图片对象引用
filename const string& 图片文件路径（相对或绝对路径）
x, y int 截取区域的左上角坐标，默认为0
w, h int 截取区域的宽高，0表示使用整个图片
newimage
静态
从窗口中捕获图片。将指定窗口区域的当前内容捕获为图片对象，可以用于截图功能或动态纹理生成。
static bool newimage ( RenderWindow& window, IMAGE& img, int x, int y, int w, int h )

参数

window RenderWindow& 源窗口引用
img IMAGE& 输出的图片对象引用
x, y int 捕获区域的左上角坐标
w, h int 捕获区域的宽高
newimage
静态
从另一图片创建新图片。可以从源图片中截取特定区域创建新图片，目标图片不保留原有的颜色滤镜，但源图片的颜色滤镜会应用到新图片上。
static bool newimage ( IMAGE& dest, IMAGE& src, int x = -1, int y = -1, int w = -1, int h = -1 )

参数

dest IMAGE& 目标图片对象引用
src IMAGE& 源图片对象引用
x, y int 截取区域的左上角坐标，-1表示从0开始
w, h int 截取区域的宽高，-1表示使用整个图片
Cutimage
静态
裁剪图片。修改现有图片对象，只保留指定矩形区域的内容。原图片的其他部分将被丢弃，图片大小会变为裁剪后的大小。
static bool Cutimage ( IMAGE& img, int x, int y, int w, int h )

参数

img IMAGE& 要裁剪的图片对象
x, y int 裁剪区域的左上角坐标
w, h int 裁剪区域的宽高
IsSmoothImage
静态
设置图片是否启用平滑处理。平滑处理（抗锯齿）可以改善图片在缩放时的视觉效果，但会略微增加渲染开销。默认启用。
static void IsSmoothImage ( IMAGE& img, bool smooth )

参数

img IMAGE& 目标图片对象
smooth bool true启用平滑处理，false禁用平滑处理
IsRepeatedImage
静态
设置图片是否平铺。当设置为平铺时，如果绘制区域大于图片本身，图片会自动重复以填充整个区域。默认不平铺。
static void IsRepeatedImage ( IMAGE& img, bool repeated )

参数

img IMAGE& 目标图片对象
repeated bool true启用平铺，false禁用平铺
SetImageColor
静态
应用图片颜色滤镜。为图片设置一个颜色滤镜，该滤镜会在图片渲染时与图片的每个像素颜色相乘。可以用于实现色调调整、颜色叠加等效果。
static void SetImageColor ( IMAGE& img, Color color )

参数

img IMAGE& 目标图片对象
color Color 要应用的颜色滤镜，RGBA格式
UpdateImage
静态
更新目标图片。将源图片的内容更新到目标图片，目标图片不保留原有的颜色滤镜，但源图片的颜色滤镜会应用到目标图片上。用于图片内容的动态更新。
static void UpdateImage ( IMAGE& img, IMAGE& dest )

参数

img IMAGE& 源图片对象
dest IMAGE& 目标图片对象
AlphaImage
静态
设置图片的透明度。通过调整图片的整体透明度（Alpha通道）实现半透明效果。alpha为0时完全透明，alpha为255时完全不透明。
static void AlphaImage ( IMAGE& img, int alpha )

参数

img IMAGE& 目标图片对象
alpha int 不透明度值，0-255
PutImage
静态
绘制图片到窗口。在指定位置绘制图片，可以指定绘制中心点坐标。这是最基本的图片绘制函数。
static void PutImage ( IMAGE& img, float x, float y, RenderWindow& Dest, float CenterX = 0.0, float CenterY = 0.0 )

参数

img IMAGE& 要绘制的图片对象
x, y float 绘制位置坐标
Dest RenderWindow& 目标窗口引用
CenterX, CenterY float 绘制中心点偏移，用于对齐和旋转参考点
PutImage
静态
绘制图片到另一图片。在目标图片的指定位置绘制源图片，目标图片不保留原有的颜色滤镜，但源图片的颜色滤镜会应用到目标图片上。
static void PutImage ( IMAGE& dest, IMAGE& src, float x, float y, float CenterX = 0.0, float CenterY = 0.0 )

参数

dest IMAGE& 目标图片对象
src IMAGE& 源图片对象
x, y float 绘制位置坐标
CenterX, CenterY float 绘制中心点偏移
PutRoteImage
静态
绘制旋转图片到窗口。在指定位置绘制旋转后的图片，可以指定旋转角度和中心点。角度单位为度。
static void PutRoteImage ( IMAGE& img, float x, float y, float angle, RenderWindow& Dest, float CenterX = 0.0, float CenterY = 0.0 )

参数

img IMAGE& 要绘制的图片对象
x, y float 绘制位置坐标
angle float 旋转角度（度）
Dest RenderWindow& 目标窗口引用
CenterX, CenterY float 旋转中心点偏移
PutScaleImage
静态
绘制缩放图片到窗口。在指定位置绘制缩放后的图片，可以分别指定宽度和高度的缩放比例。1.0表示原始大小。
static void PutScaleImage ( IMAGE& img, float x, float y, float scaleX, float scaleY, RenderWindow& Dest, float CenterX = 0.0, float CenterY = 0.0 )

参数

img IMAGE& 要绘制的图片对象
x, y float 绘制位置坐标
scaleX, scaleY float 宽度和高度的缩放比例
Dest RenderWindow& 目标窗口引用
CenterX, CenterY float 缩放中心点偏移
PutRoteScaleImage
静态
绘制旋转缩放图片到窗口。在指定位置绘制同时经过旋转和缩放变换的图片。这是最灵活的图片绘制函数，支持完整的2D变换。
static void PutRoteScaleImage ( IMAGE& img, float x, float y, float angle, float scaleX, float scaleY, RenderWindow& Dest, float CenterX = 0.0, float CenterY = 0.0 )

参数

img IMAGE& 要绘制的图片对象
x, y float 绘制位置坐标
angle float 旋转角度（度）
scaleX, scaleY float 宽度和高度的缩放比例
Dest RenderWindow& 目标窗口引用
CenterX, CenterY float 变换中心点偏移
GetWidth
静态
获取图片宽度。返回图片的像素宽度，可以用于布局计算或自适应界面设计。
static int GetWidth ( IMAGE& img )

参数

img IMAGE& 目标图片对象
GetHeight
静态
获取图片高度。返回图片的像素高度，与GetWidth配合使用可以获取图片的完整尺寸信息。
static int GetHeight ( IMAGE& img )

参数

img IMAGE& 目标图片对象
ClearImage
静态
清理图片为纯色背景。将整个图片设置为指定的纯色，可以用于清空画布或创建纯色背景图片。
static void ClearImage ( IMAGE& image, Color backcolor )

参数

image IMAGE& 目标图片对象
backcolor Color 背景颜色，RGBA格式

QImageEx 类 - 高级图片特效

ZoomImage
静态
缩放图片到指定尺寸。将图片缩放到指定的宽度和高度，会改变图片的实际像素尺寸。常用于图片的预缩放处理。
static void ZoomImage ( IMAGE& img, int scaleW, int scaleH )

参数

img IMAGE& 目标图片对象（会被修改）
scaleW int 目标宽度（像素）
scaleH int 目标高度（像素）
ZoomImage
静态
按比例缩放图片。使用缩放因子来调整图片大小，1.0表示原始大小，0.5表示缩小一半，2.0表示放大一倍。支持非均匀缩放。
static void ZoomImage ( float scaleW, float scaleH, IMAGE& img )

参数

scaleW float 宽度缩放比例
scaleH float 高度缩放比例
img IMAGE& 目标图片对象（会被修改）
BlurImage
静态
高斯模糊图片。对图片应用高斯模糊效果，可以创建柔和的阴影、景深效果或视觉焦点。sigma参数控制模糊强度，必须大于0。
static void BlurImage ( IMAGE& img, double sigma, int downscale )

参数

img IMAGE& 目标图片对象（会被修改）
sigma double 模糊程度，值越大越模糊，必须>0
downscale int 降采样倍数，提高性能但会降低质量
FogImage
静态
雾化图片效果。创建类似朦胧的效果，通过控制透明度和模糊程度来模拟朦胧感。alpha为0时完全无效，255为完全模糊。
static void FogImage ( IMAGE& img, double sigma, int downscale, int alpha )

参数

img IMAGE& 目标图片对象（会被修改）
sigma double 雾化的高斯模糊程度
downscale int 降采样倍数，提高性能但会降低质量
alpha int 不透明度，0-255，0为完全透明，255为完全不透明
FeatherImage
静态
羽化图片边缘。创建渐变的边缘效果，使图片边缘逐渐变得透明。支持三种羽化模式：平滑羽化、高斯羽化、余弦羽化。默认使用平滑羽化。
static void FeatherImage ( IMAGE& img, float strength, int downscale, int mode = 1 )

参数

img IMAGE& 目标图片对象（会被修改）
strength float 羽化强度，值越大边缘越模糊
downscale int 降采样倍数，提高性能但会降低质量
mode int 羽化模式：1-平滑羽化，2-高斯羽化，3-余弦羽化
BlackWImage
静态
黑白化图片。将彩色图片转换为黑白图片，可以控制黑白化的强度。适用于创建复古效果、强调轮廓或降低视觉复杂度。
static void BlackWImage ( IMAGE& img, float strenth, int downscale )

参数

img IMAGE& 目标图片对象（会被修改）
strenth float 黑白化强度，0-100，100为完全黑白
downscale int 降采样倍数，提高性能但会降低质量
