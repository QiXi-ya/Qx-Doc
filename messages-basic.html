底层函数

QMsg 类 - 消息处理的基础功能

UpdateMsg
静态
更新窗口消息。这个函数必须在每个主循环开始时调用，用于处理当前窗口的消息队列。它会更新所有消息状态，包括鼠标、键盘和窗口消息。
static void UpdateMsg ( RenderWindow& Dest )

参数

Dest RenderWindow& 目标窗口引用，需要更新消息的窗口
IsClose
静态
判断当前窗口是否接收到关闭消息。当用户点击窗口关闭按钮或系统发送关闭命令时，此函数返回true。通常用于控制主循环的退出条件。
static bool IsClose ()
ResetCloseMsg
静态
重置关闭消息状态。用于多窗口应用程序，当一个窗口关闭后，需要重置关闭消息状态以便正确处理其他窗口的关闭消息。
static void ResetCloseMsg ()
SetUpdateId
静态
设置全局默认的消息ID。主要用于多窗口应用程序，可以指定当前活动的消息ID。具有默认值0，表示第一个窗口。
static void SetUpdateId ( int id )

参数

id int 消息ID，用于标识不同的消息流，默认为0
SetSleepTime
静态
设置消息睡眠时间。控制消息处理循环的休眠时间，单位毫秒。可以降低CPU使用率，特别是在不需要实时响应的应用中。
static void SetSleepTime ( int time )

参数

time int 睡眠时间，单位毫秒
ClearMsg
静态
清理消息缓冲区。清除指定ID的消息缓冲区，释放内存并重置状态。在多窗口应用中，当窗口关闭时可以调用此函数清理对应消息流。
static void ClearMsg ( int id = -1 )

参数

id int 消息ID，-1表示清理所有消息缓冲区
