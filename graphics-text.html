文字显示 API

QText 类 - 字体控制、文本绘制和对齐方式

FontAdjustType 枚举

FontAdjustType 枚举用于控制文本的对齐方式，包括水平对齐和垂直对齐。通过组合不同的对齐值，可以实现灵活的文本布局。

枚举值 描述 应用场景
ADJUST_LEFT 左对齐（水平对齐） 普通段落、列表项、左侧标签
ADJUST_RIGHT 右对齐（水平对齐） 价格、时间、右侧状态栏
ADJUST_CENTER 居中对齐（水平和垂直通用） 标题、按钮文字、对话框内容
ADJUST_UP 上对齐（垂直对齐） 图标标签、顶部状态信息
ADJUST_DOWN 下对齐（垂直对齐） 脚注、底部注释、下标

对齐方式示意图

水平对齐
左对齐 | 居中对齐 | 右对齐
垂直对齐
上对齐 | 居中对齐 | 下对齐

QText 类 - 文本绘制功能

Xyprintf
静态
在指定位置绘制文本（基本版本）。使用当前设置的字体配置（颜色、大小、字体）在指定坐标绘制文本。这是最简单的文本绘制函数。
static void Xyprintf ( float x, float y, string str, RenderWindow& Dest )

参数

x, y float 文本绘制起始坐标
str string 要绘制的文本内容
Dest RenderWindow& 目标窗口引用
Xyprintf
静态
在指定位置绘制带颜色的文本。可以临时指定文本颜色，覆盖当前设置的字体颜色。适用于需要突出显示特定文本的情况。
static void Xyprintf ( float x, float y, string str, Color color, RenderWindow& Dest )

参数

x, y float 文本绘制起始坐标
str string 要绘制的文本内容
color Color 文本颜色，RGBA格式
Dest RenderWindow& 目标窗口引用
Xyprintf
静态
在指定位置绘制带颜色和大小的文本。可以临时指定文本颜色和大小，覆盖当前设置的字体配置。这是最灵活的文本绘制函数。
static void Xyprintf ( float x, float y, string str, Color color, int size, RenderWindow& Dest )

参数

x, y float 文本绘制起始坐标
str string 要绘制的文本内容
color Color 文本颜色，RGBA格式
size int 文本大小（像素高度）
Dest RenderWindow& 目标窗口引用
SetFontColor
静态
设置字体颜色。设置全局的文本绘制颜色，这个颜色会影响后续所有使用默认颜色绘制的文本，直到再次修改。
static void SetFontColor ( Color color )

参数

color Color 要设置的字体颜色，RGBA格式
SetFontSize
静态
设置字体大小。设置全局的文本绘制大小，这个大小会影响后续所有使用默认大小绘制的文本，直到再次修改。
static void SetFontSize ( int size )

参数

size int 要设置的字体大小（像素高度）
SetFontConfig
静态
同时设置字体颜色和大小。一次性设置文本绘制的颜色和大小配置，比分别调用 SetFontColor 和 SetFontSize 更高效。
static void SetFontConfig ( Color color, int size )

参数

color Color 要设置的字体颜色，RGBA格式
size int 要设置的字体大小（像素高度）
SetFont
静态
设置字体。使用已加载的 Font 对象设置文本绘制的字体。只设置字体类型，不修改颜色和大小。
static void SetFont ( Font font )

参数

font Font 要设置的字体对象
SetFont
静态
设置字体、颜色和大小。使用 Font 对象并同时设置颜色和大小，一次性完成字体的完整配置。
static void SetFont ( Font font, Color color, int size )

参数

font Font 要设置的字体对象
color Color 要设置的字体颜色，RGBA格式
size int 要设置的字体大小（像素高度）
SetFont
静态
从文件设置字体。从字体文件（如 .ttf、.otf）加载并设置字体。只设置字体类型，不修改颜色和大小。
static void SetFont ( string filename )

参数

filename string 字体文件路径（相对或绝对路径）
SetFont
静态
从文件设置字体、颜色和大小。从字体文件加载字体，并同时设置颜色和大小，一次性完成字体的完整配置。
static void SetFont ( string filename, Color color, int size )

参数

filename string 字体文件路径（相对或绝对路径）
color Color 要设置的字体颜色，RGBA格式
size int 要设置的字体大小（像素高度）
SetFontAdjust
静态
设置文本对齐方式。控制文本相对于绘制坐标的对齐方式，包括水平对齐和垂直对齐。这个设置会影响后续所有文本的绘制位置。
static void SetFontAdjust ( FontAdjustType horizontal, FontAdjustType vertical )

参数

horizontal FontAdjustType 水平对齐方式：ADJUST_LEFT、ADJUST_RIGHT、ADJUST_CENTER
vertical FontAdjustType 垂直对齐方式：ADJUST_UP、ADJUST_DOWN、ADJUST_CENTER

对齐方式示例

ADJUST_LEFT + ADJUST_UP 文本从坐标点向右下角扩展（默认）
ADJUST_CENTER + ADJUST_CENTER 文本在坐标点中心对齐
ADJUST_RIGHT + ADJUST_DOWN 文本从坐标点向左上角扩展
ADJUST_CENTER + ADJUST_UP 文本在水平居中、垂直上对齐
