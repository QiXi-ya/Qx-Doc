系统函数

QSystem 类 - 系统级功能接口和工具函数

QSystem 类

静态类

QSystem 是一个系统级功能类，提供与操作系统交互的各种功能，包括系统信息获取、窗口样式控制、时间获取、任务栏管理、随机数生成等。这些函数封装了底层的操作系统API调用，简化了系统级操作的实现。

系统信息获取函数

GetScreenSize
静态
获取屏幕的物理尺寸。返回主显示器的屏幕分辨率，以像素为单位。返回值为POINT结构，其中x成员为宽度，y成员为高度。适用于需要根据屏幕大小进行自适应布局的场景。
static POINT GetScreenSize ()

返回值

类型 说明
POINT 包含屏幕宽度和高度的POINT结构，x为宽度，y为高度（单位：像素）
IsDarkMode
静态
判断当前系统是否为深色模式。检测Windows操作系统的主题模式，判断是否启用了深色主题。适用于需要根据系统主题调整界面颜色的应用程序。
static bool IsDarkMode ()

返回值

类型 说明
bool true表示系统处于深色模式，false表示系统处于浅色模式
IsWindows11
静态
判断当前版本是否为Windows 11。检测操作系统版本，判断当前系统是否为Windows 11或更高版本。可用于实现特定于Windows 11的功能，如圆角窗口效果。
static bool IsWindows11 ()

返回值

类型 说明
bool true表示当前系统是Windows 11或更高版本，false表示是其他Windows版本

窗口样式控制函数

RoundWindowDWM
静态
圆角化窗口。通过Windows DWM（Desktop Window Manager）API设置窗口的圆角样式，此功能仅在Windows 11中有效。可以创建现代化的圆角窗口界面，提升视觉体验。
static void RoundWindowDWM ( HWND hwnd, int type )

参数

hwnd HWND 目标窗口的句柄，要应用圆角效果的窗口
type int 圆角类型：1-圆角，2-小圆角，3-方角（直角）

系统要求

操作系统 仅支持 Windows 11 或更高版本
依赖 需要系统启用DWM（桌面窗口管理器）功能
SetWindowBorder
静态
设置窗口边框的颜色。修改指定窗口的非客户区边框颜色。可以用于自定义窗口外观，创建与应用程序主题相匹配的窗口边框。
static void SetWindowBorder ( HWND hwnd, COLORREF color )

参数

hwnd HWND 目标窗口的句柄，要设置边框颜色的窗口
color COLORREF 目标颜色，Windows颜色引用（通常通过RGB宏创建）

时间获取函数

GetTimeNow
静态
获取当前时间的字符串。返回当前系统时间的格式化字符串，默认格式为"HH:MM"（24小时制，包含小时和分钟）。适用于需要显示当前时间的场景，如日志记录、状态显示等。
static std::string GetTimeNow ()

返回值

类型 说明
std::string 当前时间的格式化字符串，默认格式为"HH:MM"（如"14:30"）
GetTimeNowEx
静态
获取指定格式的当前时间的字符串。提供高度可配置的时间格式选项，可以控制时间制式（12/24小时制）、分隔符样式（冒号或空格）以及要包含的时间组件（年、月、日、时、分、秒）。
static std::string GetTimeNowEx ( bool hour12 = false, bool addcircle = true, bool year = false, bool month = false, bool day = false, bool hour = true, bool min = true, bool sec = true )

参数

hour12 bool 时间制式：true为12小时制，false为24小时制（默认）
addcircle bool 分隔符样式：true使用冒号分隔（如"12:00"），false使用空格分隔（如"12 00"）
year bool 是否包含年份，true包含，false不包含（默认）
month bool 是否包含月份，true包含，false不包含（默认）
day bool 是否包含日期，true包含，false不包含（默认）
hour bool 是否包含小时，true包含（默认），false不包含
min bool 是否包含分钟，true包含（默认），false不包含
sec bool 是否包含秒钟，true包含（默认），false不包含

返回值

类型 说明
std::string 根据参数格式化的当前时间字符串

使用示例

GetTimeNowEx(false, true, false, false, false, true, true, false) "14:30"（24小时制，冒号分隔，只包含时和分）
GetTimeNowEx(true, false, true, true, true, true, true, true) "2023 12 25 02 30 45 PM"（12小时制，空格分隔，包含所有时间组件）
GetTimeNowEx(false, true, true, true, true, false, false, false) "2023-12-25"（只包含日期）

系统控制函数

HideTaskBar
静态
隐藏或显示任务栏。控制Windows任务栏的显示状态，可以临时隐藏任务栏以创建全屏应用程序或特殊显示效果。通常用于游戏、演示软件或需要最大化屏幕空间的应用程序。
static void HideTaskBar ( bool hide )

参数

hide bool 隐藏状态：true隐藏任务栏，false显示任务栏

注意事项

权限要求 需要应用程序具有足够的系统权限
恢复 应用程序退出时应恢复任务栏显示状态
GetRand
静态
获取指定范围内的随机整数。生成一个在指定最小值和最大值之间的随机整数（包含最小值和最大值）。使用系统级随机数生成器，适用于需要随机数的各种场景，如游戏、模拟、测试等。
static int GetRand ( int min, int max )

参数

min int 随机数范围的最小值（包含）
max int 随机数范围的最大值（包含）

返回值

类型 说明
int 范围在[min, max]之间的随机整数（包含边界值）

边界条件

min > max 函数会自动交换min和max的值，确保正确范围
min == max 总是返回相同的值（min或max）
