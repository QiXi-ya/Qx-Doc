窗口管理 API

QWindow 静态类 - 窗口创建、配置和管理

QWindow 类

静态类

QWindow 是一个完全静态的类，提供了窗口创建和管理的所有方法。所有方法都是静态的，无需创建类实例即可使用。该类封装了底层的窗口操作，提供了简单易用的 API 接口。包含窗口创建、配置、位置控制、大小设置、标题设置、图标设置等功能。

CreateGraphWindow
静态
创建并初始化一个新的图形窗口。这是窗口系统的核心函数，可以自定义窗口的位置、大小、标题、样式和状态。如果坐标或大小参数为-1，将使用系统默认值。
static void CreateGraphWindow ( RenderWindow& Dest, int x = -1, int y = -1, int w = -1, int h = -1, string title = "QGraphWindow", uint32_t style = 7U, sf::State state = sf::State::Windowed, const sf::ContextSettings& setting = {} )

参数

Dest RenderWindow& 目标窗口引用，创建的窗口对象将存储在此引用中
x, y int 窗口坐标，小于0时使用系统默认位置
w, h int 窗口宽高，小于0时使用系统默认大小
title string 窗口标题，空值时使用默认标题
style uint32_t 窗口样式标志位组合
state sf::State 窗口状态（全屏、窗口化、最大化等）
setting ContextSettings& OpenGL上下文设置，用于3D渲染
DelayFps
静态
控制窗口的帧率刷新，用于实现稳定的帧率控制。这个函数在游戏循环中非常重要，可以防止程序占用过多CPU资源，同时保持平滑的动画效果。
static void DelayFps ( int fps, RenderWindow& Dest )

参数

fps int 期望的帧率上限，如60、120等
Dest RenderWindow& 目标窗口引用
IsFocus
静态
判断当前窗口是否为焦点窗口。焦点窗口是接收用户输入（键盘、鼠标）的窗口。这个函数通常用于只在窗口获得焦点时处理用户输入，优化程序性能。
static bool IsFocus ( RenderWindow& Dest )

参数

Dest RenderWindow& 目标窗口引用
SetTitle
静态
设置窗口标题。可以动态修改窗口标题栏显示的文本，适用于需要更新状态信息的应用程序，如游戏得分、文档名称等。
static void SetTitle ( RenderWindow& Dest, const String& title )

参数

Dest RenderWindow& 目标窗口引用
title const String& 新的窗口标题
SetIcon
静态
从文件中设置窗口图标。窗口图标显示在标题栏、任务栏和窗口管理器中。支持常见的图片格式，如PNG、JPG、BMP等。
static void SetIcon ( RenderWindow& Dest, const String& path )

参数

Dest RenderWindow& 目标窗口引用
path const String& 图标文件路径，相对路径或绝对路径
SetMouseCursorVisible
静态
设置鼠标在窗口上是否可见。在游戏或全屏应用中，通常需要隐藏鼠标光标，防止干扰用户视线。在编辑器中，则需要显示鼠标光标。
static void SetMouseCursorVisible ( RenderWindow& Dest, bool visible )

参数

Dest RenderWindow& 目标窗口引用
visible bool true显示鼠标，false隐藏鼠标
MoveWindow
静态
移动窗口到屏幕的指定位置。可以用于实现窗口拖拽、自动布局或屏幕位置恢复功能。坐标是相对于屏幕左上角的绝对坐标。
static void MoveWindow ( RenderWindow& Dest, int x, int y )

参数

Dest RenderWindow& 目标窗口引用
x int 新的X坐标（屏幕坐标）
y int 新的Y坐标（屏幕坐标）
SetWindowSize
静态
设置窗口大小。可以动态调整窗口的宽度和高度，适用于响应式布局、窗口缩放或全屏切换功能。注意：设置的大小不包括窗口边框。
static void SetWindowSize ( RenderWindow& Dest, int width, int height )

参数

Dest RenderWindow& 目标窗口引用
width int 新的窗口宽度（像素）
height int 新的窗口高度（像素）
GetWindowSize
静态
获取窗口的当前大小。返回包含宽度和高度的Vector2i结构。这个函数常用于自适应布局、计算纵横比或根据窗口大小调整UI元素。
static Vector2i GetWindowSize ( RenderWindow& Dest )

参数

Dest RenderWindow& 目标窗口引用
GetWindowPos
静态
获取窗口在屏幕上的当前位置。返回包含X和Y坐标的Vector2i结构。可以用于保存窗口位置配置，实现窗口位置记忆功能。
static Vector2i GetWindowPos ( RenderWindow& Dest )

参数

Dest RenderWindow& 目标窗口引用
SetWindowMinSize
静态
设置窗口的最小大小限制。防止用户将窗口缩放到过小，影响UI元素的正常显示。这个限制只影响用户拖拽调整窗口大小的操作，不限制程序代码设置更小的窗口大小。
static void SetWindowMinSize ( RenderWindow& Dest, int width, int height )

参数

Dest RenderWindow& 目标窗口引用
width int 最小宽度限制（像素）
height int 最小高度限制（像素）
