颜色与基础图形

QGraph 类 - 线条、圆形、矩形等基本图形绘制

枚举类型

枚举名称 枚举值 描述
LineType LINE_SOLID 实线
LINE_CENTER 中心线（虚线）
LINE_DOTTED 点线
LineCapType LINECAP_FLAT 平头线帽
LINECAP_ROUND 圆头线帽（默认）
LINECAP_SQUART 方头线帽

全局函数

setcolor
静态
设置当前绘制颜色。这个颜色会影响后续所有线条和图形边框的绘制，直到再次调用 setcolor 修改颜色。
static void setcolor ( Color color )

参数

color Color 要设置的颜色，通常为 RGBA 格式
setfillcolor
静态
设置当前填充颜色。这个颜色会影响后续所有填充图形的内部填充颜色，直到再次调用 setfillcolor 修改颜色。
static void setfillcolor ( Color color )

参数

color Color 要设置的填充颜色，通常为 RGBA 格式

Line 类 - 线条绘制

line
静态
绘制一条直线。从点 (x1, y1) 到点 (x2, y2) 绘制一条直线，使用当前设置的线宽、线型和颜色。
static void line ( float x1, float y1, float x2, float y2, RenderWindow& Dest )

参数

x1, y1 float 起点坐标
x2, y2 float 终点坐标
Dest RenderWindow& 目标窗口引用
lineEx
静态
绘制具有自定义粗细和样式的直线。可以指定线宽、线型（实线、虚线、点线），比基础的 line 函数提供更多控制选项。
static void lineEx ( float x, float y, float x2, float y2, int thickness, LineType type, RenderWindow& Dest )

参数

x, y float 起点坐标
x2, y2 float 终点坐标
thickness int 线条粗细（像素）
type LineType 线条类型（实线、虚线、点线）
Dest RenderWindow& 目标窗口引用
setlinewidth
静态
设置全局线条粗细。这个设置会影响后续所有线条绘制函数的线宽，直到再次调用此函数修改线宽。
static void setlinewidth ( int thickness = 1 )

参数

thickness int 线条粗细，默认值为1像素
setlinetype
静态
设置全局线条类型。可以设置为实线、虚线或点线，这个设置会影响后续所有线条绘制函数的线型。
static void setlinetype ( LineType type = LINE_SOLID )

参数

type LineType 线条类型，默认为实线
setlinecap
静态
设置线条端点样式。可以设置为平头、圆头或方头，这个设置会影响线条两端的显示效果，特别是在线宽较大时效果更明显。
static void setlinecap ( LineCapType type = LINECAP_ROUND )

参数

type LineCapType 线帽类型，默认为圆头

Circle 类 - 圆形绘制

circle
静态
绘制一个圆形边框。绘制指定中心和半径的圆形，只绘制边框，不填充内部。边框使用当前设置的颜色和线宽。
static void circle ( float x, float y, float radius, RenderWindow& Dest )

参数

x, y float 圆心坐标
radius float 圆的半径（像素）
Dest RenderWindow& 目标窗口引用
fillcircle
静态
绘制一个填充的圆形，带有边框。圆形内部使用当前填充颜色填充，边框使用当前绘制颜色。这是最常用的圆形绘制函数。
static void fillcircle ( float x, float y, float radius, RenderWindow& Dest )

参数

x, y float 圆心坐标
radius float 圆的半径（像素）
Dest RenderWindow& 目标窗口引用
fillcircle_withoutborder
静态
绘制一个填充的圆形，不绘制边框。圆形内部使用当前填充颜色填充，但没有边框。适用于需要纯色圆形的情况。
static void fillcircle_withoutborder ( float x, float y, float radius, RenderWindow& Dest )

参数

x, y float 圆心坐标
radius float 圆的半径（像素）
Dest RenderWindow& 目标窗口引用
ellipse
静态
绘制一个椭圆边框。绘制指定中心、横向半径和纵向半径的椭圆，只绘制边框，不填充内部。适用于需要椭圆形状的图形。
static void ellipse ( float x, float y, float radiusX, float radiusY, RenderWindow& Dest )

参数

x, y float 椭圆中心坐标
radiusX float 横向半径（像素）
radiusY float 纵向半径（像素）
Dest RenderWindow& 目标窗口引用
fillellipse
静态
绘制一个填充的椭圆，带有边框。椭圆内部使用当前填充颜色填充，边框使用当前绘制颜色。可以控制椭圆的横向和纵向半径。
static void fillellipse ( float x, float y, float radiusX, float radiusY, RenderWindow& Dest )

参数

x, y float 椭圆中心坐标
radiusX float 横向半径（像素）
radiusY float 纵向半径（像素）
Dest RenderWindow& 目标窗口引用
fillellipse_withoutborder
静态
绘制一个填充的椭圆，不绘制边框。椭圆内部使用当前填充颜色填充，但没有边框。适用于需要纯色椭圆的情况。
static void fillellipse_withoutborder ( float x, float y, float radiusX, float radiusY, RenderWindow& Dest )

参数

x, y float 椭圆中心坐标
radiusX float 横向半径（像素）
radiusY float 纵向半径（像素）
Dest RenderWindow& 目标窗口引用
arc
静态
绘制圆弧。绘制指定中心、半径、起始角度和绘制角度的圆弧。只绘制圆弧的边框，不填充。角度通常以度为单位。
static void arc ( float x, float y, float radius, float startAngle, float endAngle, RenderWindow& Dest )

参数

x, y float 圆弧中心坐标
radius float 圆弧半径（像素）
startAngle float 起始角度（度）
endAngle float 绘制角度（度），从起始角度开始计算
Dest RenderWindow& 目标窗口引用
fillarc
静态
绘制填充的圆弧，不绘制边框。绘制指定参数的填充扇形区域，内部使用当前填充颜色填充，但没有边框。适用于绘制扇形图等场景。
static void fillarc ( float x, float y, float radius, float startAngle, float endAngle, RenderWindow& Dest )

参数

x, y float 圆弧中心坐标
radius float 圆弧半径（像素）
startAngle float 起始角度（度）
endAngle float 绘制角度（度），从起始角度开始计算
Dest RenderWindow& 目标窗口引用

Rectangle 类 - 矩形绘制

rect
静态
绘制矩形边框。绘制指定位置和大小的矩形，只绘制边框，不填充内部。边框使用当前设置的颜色和线宽。
static void rect ( float x, float y, float w, float h, RenderWindow& Dest )

参数

x, y float 矩形左上角坐标
w float 矩形宽度（像素）
h float 矩形高度（像素）
Dest RenderWindow& 目标窗口引用
fillrect
静态
绘制填充的矩形，带有边框。矩形内部使用当前填充颜色填充，边框使用当前绘制颜色。这是最常用的矩形绘制函数。

参数

x, y float 矩形左上角坐标
w float 矩形宽度（像素）
h float 矩形高度（像素）
Dest RenderWindow& 目标窗口引用
fillrect_withoutborder
静态

static void fillrect_withoutborder ( float x, float y, float w, float h, RenderWindow& Dest )

参数

x, y float 矩形左上角坐标
w float 矩形宽度（像素）
h float 矩形高度（像素）
Dest RenderWindow& 目标窗口引用
roundrect

static void fillroundrect_withoutborder ( float x, float y, float w, float h, float radius, RenderWindow& Dest )
