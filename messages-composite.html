复合函数

QMsg::Composite 类 - 高级消息处理和辅助功能

复合函数概述

QMsg::Composite 类提供了一系列高级消息处理功能，将常见的消息处理模式封装为简单易用的接口。包括区域滚轮控制、鼠标状态组合检测、键盘输入辅助等功能。这些函数简化了复杂交互逻辑的实现，提高了开发效率。

鼠标与滚轮组合

GetRectWheel 和 IsMouseInRectDown 函数，简化区域交互的实现。

键盘输入辅助

GetKeyPressNum 和 GetKeyPressString 函数，方便获取按键状态。

辅助工具函数

UpdateWheel 和 UpdatePress 函数，提供滚轮和输入处理辅助。

鼠标与滚轮组合函数

GetRectWheel
静态
获取一个区域内鼠标的滚轮值。当鼠标在指定矩形区域内滚动时，返回滚轮的当前值（通常是120的倍数）；如果鼠标不在区域内则返回0。正值表示向上滚动，负值表示向下滚动。
static int GetRectWheel ( RenderWindow& Dest, int x, int y, int w, int h, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用
x, y int 矩形区域的左上角坐标（窗口坐标）
w, h int 矩形区域的宽度和高度（像素）
id int 消息ID，-1表示使用默认消息ID
IsMouseInRectDown
静态
判断当前鼠标是否在一个区域内并且处于按下状态。这个函数结合了区域检测和按钮状态检测，通常用于实现按钮点击、区域选择等交互功能。只有当鼠标在指定区域内且指定按钮被按下时返回true。
static bool IsMouseInRectDown ( RenderWindow& Dest, Mouse::Button button, int x, int y, int w, int h, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用
button Mouse::Button 要检测的鼠标按钮（左键、右键、中键等）
x, y int 矩形区域的左上角坐标（窗口坐标）
w, h int 矩形区域的宽度和高度（像素）
id int 消息ID，-1表示使用默认消息ID

键盘输入辅助函数

GetKeyPressNum
静态
获取键盘按下的数字键。检测当前按下的键是否是数字键（0-9），如果是则返回对应的数字值（0-9）；如果不是数字键或者没有按键被按下，返回-1。适用于需要数字输入的场景。
static int GetKeyPressNum ( RenderWindow& Dest, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用，检测此窗口的键盘事件
id int 消息ID，-1表示使用默认消息ID
GetKeyPressString
静态
获取键盘按下的键的字符串。返回当前按下的按键对应的字符串表示，如"A"、"Space"、"Escape"等。如果没有按键被按下，则返回空字符串。可以用于获取用户按下的具体按键。
static string GetKeyPressString ( RenderWindow& Dest, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用，检测此窗口的键盘事件
id int 消息ID，-1表示使用默认消息ID

辅助工具函数

UpdateWheel
静态
更新鼠标滚轮值，此为辅助函数。管理滚轮值的自动衰减和范围限制。可以用于实现平滑的滚轮滚动效果，如列表滚动、缩放控制等。函数会自动将滚轮值限制在指定范围内，并应用速度衰减。
static void UpdateWheel ( int& wheel, int& tempspeed, const int speed, const int max, const int min )

参数

wheel int& 滚轮值变量引用，会被函数更新
tempspeed int& 临时速度变量引用，用于存储衰减状态
speed const int 衰减速度，值越大衰减越快
max const int 滚轮最大值限制
min const int 滚轮最小值限制

使用说明

工作原理 通过 speed 参数控制滚轮值的衰减速度，将 wheel 值限制在 [min, max] 范围内
调用时机 应在主循环中每帧调用，实现平滑的滚动效果
应用场景 滚动列表、缩放视图、数值调整器等需要平滑滚动的场景
UpdatePress
静态
处理键盘输入的字符串，此为辅助函数，主要用于输入框等功能。自动处理键盘输入，包括字符添加、退格删除、回车确认等。支持数字模式，可以限制只输入数字字符。
static void UpdatePress ( string& str, bool BanAcl = false, int id = -1 )

参数

str string& 要更新的字符串引用，函数会修改此字符串
BanAcl bool 是否只允许数字输入，true表示只允许数字字符
id int 消息ID，-1表示使用默认消息ID

功能特性

字符处理 自动添加可打印字符到字符串末尾
退格功能 按Backspace键删除最后一个字符
回车确认 按Enter键不修改字符串，可配合其他逻辑处理确认
数字限制 BanAcl为true时，只允许输入数字字符0-9
