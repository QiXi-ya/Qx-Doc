颜色处理

QColor 类 - 颜色处理和转换功能

QColor 类

静态类

QColor 类提供了丰富的颜色处理和转换功能，包括颜色灰度化、亮度调整、透明度控制、颜色对比和判断等。这些函数简化了颜色的处理逻辑，方便开发者实现各种视觉效果。

GrayColor
静态
灰度化颜色。将指定颜色转换为灰度颜色，可以控制灰度化的程度。灰度值为0时返回原色，灰度值为100时返回完全灰度。
static Color GrayColor ( const Color& color, float percent )

参数

color const Color& 目标颜色，要转换为灰度的颜色
percent float 灰度值，范围 [0 - 100]。0为原色，100为完全灰度

返回值

类型 说明
Color 经过灰度处理后的颜色
DarkColor
静态
暗化颜色。降低指定颜色的亮度，使其变暗。暗度值为0时返回原色，暗度值为100时返回接近黑色的颜色。
static Color DarkColor ( const Color& color, float percent )

参数

color const Color& 目标颜色，要变暗的颜色
percent float 暗度值，范围 [0 - 100]。0为原色，100为完全变暗

返回值

类型 说明
Color 经过暗度处理后的颜色
LightColor
静态
亮化颜色。提高指定颜色的亮度，使其变亮。亮度值为0时返回原色，亮度值为100时返回接近白色的颜色。
static Color LightColor ( const Color& color, float percent )

参数

color const Color& 目标颜色，要变亮的颜色
percent float 亮度值，范围 [0 - 100]。0为原色，100为完全变亮

返回值

类型 说明
Color 经过亮度处理后的颜色
AlphaColor
静态
透明化颜色。修改指定颜色的透明度（Alpha通道）。可以创建半透明或完全透明的颜色效果，适用于实现淡入淡出、阴影叠加等视觉效果。
static Color AlphaColor ( const Color& color, uint8_t alpha )

参数

color const Color& 目标颜色，要修改透明度的颜色
alpha uint8_t 透明度值，范围 [0 - 255]。0为完全透明，255为完全不透明

返回值

类型 说明
Color 经过透明度处理后的颜色
IsDarkColor
静态
判断是否为深色。通过计算颜色的亮度值，判断指定颜色是否属于深色系。适用于根据背景色自动调整文字颜色等场景。
static bool IsDarkColor ( const Color& color )

参数

color const Color& 要判断的颜色

返回值

类型 说明
bool true表示是深色，false表示是浅色
IsColorSame
静态
判断颜色是否相似或者相同。可以精确对比两个颜色，也可以进行相似性比较。支持控制是否对比Alpha通道，以及相似性容忍度。
static bool IsColorSame ( const Color& colorA, const Color& colorB, bool UseAlpha = false, bool isExact = false, int VagueValue = 10 )

参数

colorA const Color& 目标颜色A
colorB const Color& 目标对比颜色B
UseAlpha bool 是否对比Alpha通道，true表示对比，false表示不对比（默认）
isExact bool 是否精确对比，true表示精确对比，false表示相似对比（默认）
VagueValue int 相似差度，用于相似对比时的容忍度（默认10）

使用模式

精确对比模式 isExact=true，要求RGB（和可选Alpha）完全相等
相似对比模式 isExact=false，允许颜色在VagueValue范围内相似

返回值

类型 说明
bool true表示相同，false表示不同
AdjustColorBright
静态
调整颜色亮度。对指定颜色进行亮度调整，可以增加或减少亮度。正值增加亮度，负值减少亮度。适用于动态调整界面元素亮度。
static Color AdjustColorBright ( const Color& color, int brightness )

参数

color const Color& 目标颜色，要调整亮度的颜色
brightness int 亮度阈值，正值增加亮度，负值减少亮度

返回值

类型 说明
Color 经过亮度处理后的颜色
