键盘消息

QMsg::KeyMsg 类 - 键盘事件处理和按键状态检测

Keyboard::Key 枚举

在 Qx 图形库中，键盘按键通过 Keyboard::Key 枚举表示。这个枚举位于 SFML 图形库中，提供了标准的键盘按键标识。以下是常用的按键枚举值：

类别 按键枚举 说明
字母键 Keyboard::Key::A 到 Keyboard::Key::Z 26个字母按键
数字键 Keyboard::Key::Num0 到 Keyboard::Key::Num9 主键盘区数字键 0-9
功能键 Keyboard::Key::F1 到 Keyboard::Key::F12 功能键 F1-F12
控制键 Keyboard::Key::Escape, Keyboard::Key::Tab, Keyboard::Key::Enter
Keyboard::Key::Space, Keyboard::Key::BackSpace, Keyboard::Key::Delete		 常用控制键
修饰键 Keyboard::Key::LShift, Keyboard::Key::RShift
Keyboard::Key::LControl, Keyboard::Key::RControl
Keyboard::Key::LAlt, Keyboard::Key::RAlt		 Shift、Ctrl、Alt 修饰键（左右区分）
方向键 Keyboard::Key::Left, Keyboard::Key::Right
Keyboard::Key::Up, Keyboard::Key::Down		 方向键
数字小键盘 Keyboard::Key::Numpad0 到 Keyboard::Key::Numpad9
Keyboard::Key::Add, Keyboard::Key::Subtract, Keyboard::Key::Multiply, Keyboard::Key::Divide		 数字小键盘按键

基本按键检测函数

IsKeyDown
静态
检测按键是否被按下。判断指定按键在当前帧是否被按下（按下事件）。这个函数只在按键刚刚按下时返回 true，适合检测单次按键事件。
static bool IsKeyDown ( RenderWindow& Dest, Keyboard::Key key, int id = -1 )

参数

Dest RenderWindow& 目标窗口引用，检测此窗口的键盘事件
key Keyboard::Key 要检测的键盘按键（如 Keyboard::A、Keyboard::Space 等）
id int 消息ID，-1表示使用默认消息ID
IsKeyUp
静态
判断当前键盘的一个按键是否抬起。检测指定按键是否在当前帧被释放（抬起事件）。这个函数只在按键刚刚抬起时返回 true。
static bool IsKeyUp ( RenderWindow& Dest, string key = "NULL", int id = -1 )

参数

Dest RenderWindow& 目标窗口引用，检测此窗口的键盘事件
key string 虚拟键码字符串表示，如"A"、"Space"、"Escape"等，"NULL"表示任意键
id int 消息ID，-1表示使用默认消息ID

高级输入处理函数

GetPressKeyList
静态
获取当前按下的按键列表。返回一个包含当前所有被按下按键的向量（vector）。可以用于检测多个按键同时按下的情况。
static vector<string> GetPressKeyList ( int id = -1 )

参数

id int 消息ID，-1表示使用默认消息ID
Keystate
静态
判断虚拟键码对应的按键是否被按下。通过Windows虚拟键码（VK）直接检测按键状态，不依赖窗口消息循环。适用于需要实时检测系统级按键状态的场景。
static bool Keystate ( int VK )

参数

VK int Windows虚拟键码，如0x41表示'A'键
